# Mané — Bios (Instagram)
Projeto React + Vite (TypeScript) para páginas de bio mobile-first.

## Rodar local
```bash
npm install
npm run dev
```
Acesse:
- `/ac` → Águas Claras (pronto)
- `/sp` → São Paulo (placeholder)
- `/bsb` → Brasília (placeholder)

## Deploy
Vercel/Netlify prontos. Inclua `public/config/` nos artefatos.

## Config por cidade
Edite `public/config/<cidade>.json` (existe `ac.json`, `sp.json`, `bsb.json`).

## Extras
- Meta Pixel `1160688802149033` já envia `PageView`.
